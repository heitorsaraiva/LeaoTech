Elementos do tipo INLINE BLOCK - Ele permite que um elemento fique na mesma linha que outrosmas ainda pode definir largura, altura, margens e padding (como um elemento block). É útil quando se deseja controlar o tamanho de elementos enquanto mantém o layout em linha.



Elementos do tipo NONE - É uma propriedade do CSS que oculta completamente um elemento da página, removendo-o do fluxo do layout



Elemento do tipo de FLEX - é uma propriedade do CSS que ativa o Flexbox, um sistema de layout que facilita o alinhamento e a distribuição de elementos dentro de um contêiner. Ele organiza os itens de forma eficiente, permitindo controle sobre a direção, alinhamento, espaçamento e flexibilidade dos elementos, independentemente de seu tamanho.
